# Roadrunner - tiny and fast

## Installation

```posh
git clone https://github.com/astares/roadrunner.git
make
```
![Roadrunner](www/pic/roadrunner.png)

## Compilation

Run 
```console
make
```
to compile all or 
 
```console
make rebuild
```
to fully rebuild.

## Run
Run 
```console
./bin/rrvm
```
to execute
